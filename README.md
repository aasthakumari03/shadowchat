# ShadowChat

ShadowChat is a real-time, dark-themed chat application with a unique QR code session-sharing feature.

## Features
- **Real-time Chat**: Powered by Socket.io.
- **Dark UI**: Styled with Tailwind CSS.
- **Session-based Messaging**: Create or join anonymous chat sessions.
- **QR Sharing**: Glowy QR popup for easy session access.

## Quick Start

### 1. Install Dependencies
```bash
npm install
```

### 2. Run the Server
```bash
npm start
```

### 3. Open the Client
Open `frontend/index.html` in your browser.

## Tech Stack
- **Backend**: Node.js, Express, Socket.io
- **Frontend**: HTML5, Tailwind CSS, Socket.io Client
